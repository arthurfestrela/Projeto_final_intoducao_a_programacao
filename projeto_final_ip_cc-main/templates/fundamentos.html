Fundamentos da Programação em Python

  • Estruturas de seleção: if, elif, else. Exemplo: if x > 10: print("Maior que 10")
  • Estruturas de repetição: for, while. Exemplo: for i in range(5): print(i)
  • Vetores e matrizes: listas e listas de listas. Exemplo: matriz = [[1,2], [3,4]]
  • Funções e procedimentos: def nome_funcao(): ...
  • Tratamentos de exceção: try, except. Exemplo: try: ... except: ...
